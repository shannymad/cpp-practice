# Linear ALgebra Lib 

A C++17 lightweight Linear Algebra Lib.Designed for educational purposes and understanding the basics of matrix computations.

## What is implemented?
- `Core operations`: Matrix addition, subtraction, and scalar multiplication.
Matrix-matrix multiplication with dimension checking.
Transposition and element-wise access with bounds checking
- `Basic algorithms`: Determinant calculation via Gaussian elimination.
LU Decomposition (with partial pivoting).Linear System Solver using forward/backward substitution.
Matrix Inversion via Gauss-Jordan elimination.

## Requirements
- C++17 compiler (GCC ≥ 9 or Clang ≥ 10)
- CMake ≥ 3.15
